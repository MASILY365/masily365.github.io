Hexo 搭建自己的博客

Hexo 是一个快速、简洁且高效的博客框架，可以一键部署到 GitHub Page 等平台。

本文主要记录通过 Hexo 并配置高颜值 icarus 主题搭建个人博客的过程。

Hexo 优点： 速度快、支持 Markdown、部署方便、可扩展性好等。

Github Page 配置

  • 新建 <username>.github.io 仓库
  • 在 <username>.github.io -> setting -> Pages 中 source 中选择 root 点击 save
  • 访问 <username>.github.io 测试

配置 GitHub Action

GitHub Actions 可以很方便实现 CI/CD 工作流。

  • 用途： 自动化测试、打包、部署等操作。
  • 支持容器： Ubuntu、Windows 和 MacOS 等系统，在容器中我们可以像在本地电脑中进行一系列操作。
  • 本文说明： 利用 Github Actions 实现自动部署 hexo 到 Github Pages。
  1. 生成部署密钥

文件路径：当前目录下会有 github-deploy-key 和 github-deploy-key.pub 两个文件

  1. 配置部署密钥

  • 私钥（github-deploy-key）：存放在博客源代码仓库

  • 复制 github-deploy-key 文件内容，在 blog 仓库 Settings -> Secrets-> Action -> Add a new secret 页面上添加。

  • 在 Name 输入框填写 HEXO_DEPLOY_PRI。

  • 在 Value 输入框填写 github-deploy-key 文件内容

  • 公钥（github-deploy-key.pub）：存在账户公钥中。

  • 复制 github-deploy-key.pub 文件内容，在 <username>.github.io 仓库 Settings -> Deploy keys -> Add deploy key 页面上添加。

  • 在 Title 输入框填写 HEXO_DEPLOY_PUB。

  • 在 Key 输入框填写 github-deploy-key.pub 文件内容。

  • 勾选 Allow write access 选项。
